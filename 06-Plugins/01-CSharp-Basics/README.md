# C# Basics

## Overview

C# is the primary programming language used to develop Microsoft Dynamics 365 CRM Plugins. Before learning plugin development, it is important to understand 
C# fundamentals such as methods, parameters, loops, arrays, class libraries, and object-oriented programming concepts.

---

## Objectives

After completing this topic, you will be able to:

- Install Visual Studio
- Create a Console Application
- Understand Namespace, Class, and Method structure
- Create and consume methods
- Work with parameterized methods
- Use arrays and loops
- Create a Class Library
- Generate a DLL
- Add project references
- Create objects and consume reusable methods

## Best Practices

- Use PascalCase for Classes and Methods.
- Use camelCase for Variables and Parameters.
- Keep methods reusable.
- Avoid duplicate code.
- Separate reusable logic into Class Libraries.
- Follow proper project structure.
