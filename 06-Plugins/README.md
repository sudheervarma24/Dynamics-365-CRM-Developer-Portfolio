# Plugins

## Overview

Plugins are server-side C# components that execute within the Microsoft Dynamics 365 CRM event pipeline to implement custom business logic, validations, and automation.

## Topics Covered

- C# Basics
- Visual Studio Setup
- Plugin Registration
- Populate Full Name Plugin
- SSC Marks Calculation Plugin
- Copy Source To Target Plugin
- Plugin Execution Pipeline
- Plugin Images
- Plugin Trace Logs
- Retrieve & RetrieveMultiple
- Exception Handling
- Audit History
- Plugin Debugging

## Learning Objectives

After completing this section, you will be able to:

- Develop plugins using C#
- Register plugins in Dynamics 365 CRM
- Understand the plugin execution pipeline
- Work with Plugin Images
- Retrieve Dataverse records
- Handle exceptions
- Debug plugins effectively
